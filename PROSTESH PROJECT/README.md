# Bun Rush — End-to-End Restaurant Analytics System 🚀

## Project Overview

Bun Rush is my first complete end-to-end data analytics and business intelligence project.

The project started with building a fictional burger restaurant website from scratch using:

- HTML
- CSS
- JavaScript

The website allows customers to:
- Browse menu items
- Add products to cart
- Place orders
- Track their orders using Order ID

---

## Admin Dashboard

I also built a complete admin dashboard that allows restaurant managers to:

- View all orders
- Update order status
- Track preparing and delivered orders
- Export orders as CSV files
- Export product lists and categories

---

## Firebase Integration

The entire website is connected to:

- Firebase Firestore
- Firebase Authentication

All customer orders are automatically stored in the cloud database.

---

## Power BI Dashboards

The project includes:
- Sales Dashboard

The dashboards analyze:
- Total Sales
- Profit Margin
- Revenue Trends
- Product Performance
- Customer Insights
- Orders Status

## Fake Orders Generator

I created a JavaScript-based script that generated more than 800 fake orders automatically to simulate realistic business data for analytics and dashboard development.
