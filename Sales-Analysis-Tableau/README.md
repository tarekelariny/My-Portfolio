# 📊 Sales Analysis Dashboard | Tableau

## 📌 Project Overview
This Tableau dashboard analyzes sales performance across different dimensions including sales trends, countries, cities, employees, and product categories.

The dashboard helps identify business performance patterns and supports data-driven decision-making.

---

## 🚀 Tools Used
- Tableau
- Excel
- Data Visualization
- Dashboard Design

---

## 📈 Key Insights
- Total Sales reached 970K
- Net Profit achieved 571K
- Profit Margin reached 59%
- Strong sales contribution from major cities
- Stable monthly sales performance throughout the year

---

## 📊 Dashboard Features
- KPI Overview
- Monthly Sales Trend
- Sales by Country
- Sales by City
- Sales by Employee
- Sales by Category
- Order Type Analysis
