# Bun Rush — End-to-End Restaurant Analytics System

An end-to-end restaurant ordering and business intelligence project that combines web development, Firebase cloud integration, data simulation, Power Query transformation, and Power BI dashboard analytics.

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Firestore
- Firebase Authentication
- Power Query
- Power BI
- DAX
